# mtc2025-be
Backend for MTC 2025

# Versioning
Python - 3.12.8
Flask - 3.1.0

# How To Start
1. Open terminal/cmd in project directory
2. Run ```pip install -r ./requirements.txt```
3. Subtitute values in main.py with actual localhost URL
- DB_HOST
- DB_PORT
- DB_USER
- DB_PASS
4. Ensure PostgreDB has database with name 'mtc'
5. Run main.py
